PiCam-Setup

A simple install script for initializing your Raspberry Pi with mpeg-streamer to stream your webcam on a Raspberry PI. To run the script make sure your repository is up-to-date and Git is installed. 

sudo apt-get update
sudo apt-get upgrade -y
sudo apt-get install -y git

After this you can download the script

git clone git://github.com/Ploeper/PiCam-Setup.git

Execute the script

cd PiCam-Setup
sudo chmod +x /picam_setup.sh
sudo ./picam_setup.sh
